Chinese Poker Rules

This game is set for 4 players. Each player receives a 13-card hand from a standard 52-card deck. Each player then divide their cards into 3 poker hands : - the first hand will contain 3 cards - the middle hand will contain 5 cards - the last hand will contain 5 cards When setting your 3 poker hands, keep in mind that: - for the first hand, Straight and Flush do not count : are only allowed High Card, One pair, and Three of a Kind - the first hand value must be lower than the middle hand value, thus making the first hand the lowest-ranking hand - the middle hand value must be lower than the last hand value, thus making the last hand the highest-ranking hand

Mis-set/invalid hands examples :

First hand Middle hand Last hand
The first hand value is greater than the middle hand value 3♥ 3♦ 3♠ 7♥ 7♦ 8♠ 8♣ A♦ 9♠ 8♠ 6♥ 5♣ 4♦
The middle hand value is greater than the last hand value A♥ A♠ 3♥ J♣ 8♣ 6♣ 3♣ 2♣ 5♦ 5♥ 5♣ K♠ Q♠

At showdown, if a player has mis-set theirs hands, the player won't be able to score any points

Showdown and Scoring

Once every player are ready, the player hands are compared (AKA the showdown) : - the player first hands are compared, and the player with the best first hand scores a point - the player middle hands are compared, and the player with the best middle hand scores a point - the player last hands are compared, and the player with the best last hand scores a point In the case of a draw between players, no one scores a point. if the same player scores 2 or 3 points, the player scores an extra point (AKA a scoop).
